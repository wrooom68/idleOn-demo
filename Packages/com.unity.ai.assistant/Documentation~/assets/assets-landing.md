# Work with assets in Unity Editor

Use Assistant and Generators in the Unity Editor to create and modify assets.

This section focuses on workflows that connect generated assets with GameObjects and Unity Editor tools. It explains how to create assets within the Unity Editor, assign them to scene objects, and interact with existing assets using natural language.
