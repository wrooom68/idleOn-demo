# Create 3D objects and terrain layers

Generate simple 3D props and terrain materials for your scenes with Generators and Assistant.

This section covers how to create single-part 3D objects from text prompts or reference images, and how to generate terrain layer materials for Unity Terrain components. You can generate assets directly through the Generators window for more control or use Assistant to guide the process with natural-language prompts and project context.

| Topic | Description |
| ----- | ----------- |
| [Create 3D objects](xref:3d-generator-overview) | Generate simple 3D props from text prompts or reference images and add them to your scene as prefabs. |
| [Create terrain layers](xref:terrain-overview) | Generate and apply terrain layer materials with compatible maps for use on Unity Terrain objects. |
