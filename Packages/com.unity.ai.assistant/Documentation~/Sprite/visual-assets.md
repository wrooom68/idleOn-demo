# Create visual assets

Generate 2D visual assets with Generators.

This section covers how to create and refine sprites, spritesheets, textures, materials, and cubemaps. You can generate assets from prompts or reference images, edit generated results, and manage them within your project.

| Topic | Description |
| ----- | ----------- |
| [Create sprites](xref:sprite-overview) | Generate single sprites or spritesheets from prompts or reference images, and refine results directly in the Unity Editor. |
| [Create textures](xref:texture2d-overview) | Generate textures from prompts or images, and adjust resolution and style settings. |
| [Create materials](xref:material-overview) | Create materials from prompts and automatically generate associated texture maps for use in shaders. |
| [Create cubemaps](xref:cubemap-overview) | Generate skyboxes and environment cubemaps from prompts or panoramic inputs. |
| [Create 3D objects and terrain layers](xref:3d-terrain-landing) | Generate 3D models and terrain textures, and apply them to scenes and Terrain components in the Unity Editor. |
