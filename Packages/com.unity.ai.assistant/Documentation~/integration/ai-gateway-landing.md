# AI Gateway

Use AI Gateway to run third-party coding agents inside the Unity Editor.

AI Gateway lets you select and use third‑party coding agents, such as [Claude Code](https://code.claude.com/docs/en/overview), [Codex](https://developers.openai.com/codex/cli/), [Gemini](https://geminicli.com/docs/get-started/installation/), and [Cursor](https://cursor.com/download), in the Assistant window. You can change agents within the Unity Editor, authenticate each agent with your own API keys, and use provider‑specific models and commands in the Assistant workflow.

When you enable AI Gateway, these agents can run actions in the Unity Editor, such as creating scripts, inspecting console logs, and modifying project content.
