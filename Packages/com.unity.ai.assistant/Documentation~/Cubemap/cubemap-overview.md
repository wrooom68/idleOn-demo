# Create cubemaps

Generate cubemaps to create realistic skyboxes, reflections, and lighting environments in Unity.

Use Cubemap Generator to create seamless, high-quality [cubemaps](xref:um-class-cubemap-landing) from text-based prompts or reference images. You can generate equi-rectangular maps to use as cubemaps, skyboxes, or reflection textures to enhance environmental lighting and mood.

| [Create a cubemap from a reference image](xref:reference-cubemap) | Use a reference image to influence cubemap style and composition. |
| [Edit a cubemap](xref:modify-cubemap) | Refine or upscale the generated cubemap. |
