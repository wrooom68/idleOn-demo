# Edit audio with Assistant

Modify audio clips directly in the Unity Editor using natural-language prompts.

This section explains how Assistant edits audio clips by removing silence, trimming specific time ranges, adjusting volume levels, and generating seamless loops. You can clean recordings, extract useful segments, balance sound levels, and create looping ambience without using external audio-editing tools. Assistant preserves your original file and generates a new audio asset for each modification.
